# Yellowbrickroad

Website project for Yellowbrickroad, a headhunting agency from Warsaw

[Link to demo](https://makspara.github.io/yellowbrickroad/)

Used technologies:
- React
- Tailwind CSS
- DaisyUI
- Vite

To run the project `git clone`, navigate to repo, `npm i` and `npm run dev`
